# Public Assets

Place your logo image here as `logo.png`.

## Logo Requirements

- **File name**: `logo.png`
- **Recommended size**: 32x32px to 128x128px (square)
- **Format**: PNG with transparency (preferred) or JPG
- **Location**: `/public/logo.png`

The logo will automatically appear in all headers across the website. If no logo is provided, a fallback "NYGM" badge will be displayed instead.
