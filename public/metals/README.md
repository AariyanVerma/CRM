# Metal Icons

Place metal icon images in this directory:

- `gold.png` - Gold metal icon/image
- `silver.png` - Silver metal icon/image  
- `platinum.png` - Platinum metal icon/image

## Image Requirements:
- Format: PNG (with transparency preferred)
- Recommended size: 128x128px or larger
- Should be square or circular icons representing each metal
- Transparent background recommended for best appearance

## Fallback:
If images are not found, the system will display a colored circle with the first letter of the metal name (G, S, P).
