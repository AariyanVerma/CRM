# PWA Icons

This PWA requires two icon files:

1. **icon-192.png** - 192x192 pixels (for Android home screen)
2. **icon-512.png** - 512x512 pixels (for splash screens and larger displays)

## Current Status

Placeholder icons are needed. You mentioned you'll load icons later.

## How to Add Icons

1. Create or obtain your app icons in the required sizes (192x192 and 512x512)
2. Save them as `icon-192.png` and `icon-512.png` in the `public` folder
3. The PWA will automatically use them

## Quick Solution (Temporary)

If you want to test the PWA immediately, you can:
- Copy your `logo.png` and resize it to 192x192 and 512x512
- Save them as `icon-192.png` and `icon-512.png` in the `public` folder

## Icon Requirements

- **Format**: PNG
- **Sizes**: 192x192 and 512x512 pixels
- **Background**: Should work on both light and dark backgrounds
- **Shape**: Square (will be automatically masked on some platforms)
